# next-chakra-redux-template
This is a next.js + chakra-ui + react-redux quick start template.

Use nextjs-13 appDir.

# how to use
1. Clone this repository.
```sh
git clone <this_repo>
```

2. cd and remove old git.
```sh
cd next-chakra-redux-template && rm -rf .git
```

3. install & start
```sh
pnpm i

# OR
yarn

# OR
npm install
```
